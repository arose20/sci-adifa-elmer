Code Documentation¶

blog.create()¶

Create a new post for the current user.

blog.delete(id)¶

Delete a post.

Ensures that the post exists and that the logged in user is the author of the post.

blog.get_post(id, check_author=True)¶

Get a post and its author by id.

Checks that the id exists and optionally that the current user is the author.

Parameters

  • id – id of post to get

  • check_author – require the current user to be the author

Returns

the post with author information

Raises

  • 404 – if a post with the given id doesn’t exist

  • 403 – if the current user isn’t the author

blog.index()¶

Show all the posts, most recent first.

blog.update(id)¶

Update a post if the current user is the author.
